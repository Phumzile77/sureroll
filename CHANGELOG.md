# Changelog

All notable changes made to the Sureroll website project are recorded in this file.

## 2026-05-27

### Added
- Added a professional blue and grey visual theme across the website.
- Added a larger blue version of the Sureroll logo: `img/Logo2-blue.png`.
- Added consistent headers, navigation menus, and footers across all five pages.
- Added responsive layouts for desktop, tablet, and mobile screen sizes.
- Added improved call-to-action sections to guide users to the contact page.
- Added improved contact and quote request forms on the Contact page.
- Added a styled location section with an embedded Google Map.

### Changed
- Updated `index.html` with a stronger homepage hero section, service overview, benefits section, and call-to-action area.
- Updated `about.html` with a professional page hero, company overview, mission, promise, and focus sections.
- Updated `services.html` with structured service cards for payroll, accounting, tax, reporting, HR administration, and full package support.
- Updated `blog.html` with a cleaner blog layout and expandable article sections.
- Updated `contactus.html` with improved form structure, labels, placeholders, and layout.
- Replaced the green logo with the blue logo on all pages to match the website colour scheme.
- Increased the logo display size in the website header.
- Moved styling into `style.css` instead of using inline styles inside the HTML pages.

### Fixed
- Fixed missing closing HTML tags on the Contact page.
- Fixed incorrect special characters caused by encoding issues.
- Fixed inconsistent page layouts between Home, About, Services, Blog, and Contact pages.
- Fixed image references so all local images used by the website point to existing files.
- Improved mobile responsiveness so content stacks neatly on smaller screens.
